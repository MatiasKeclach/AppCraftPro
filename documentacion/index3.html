Tienda Virtual "Las Gladiolas"

Presentación de arquitectura, diseño y funcionamiento para el cliente

1. Resumen de la Arquitectura

La tienda virtual está diseñada para ser escalable, SEO friendly y en tiempo real. Se compone de:

  • Base de datos ligera y rápida con BetterSQLite3.
  • Backend con Node.js + Express para manejo de APIs REST y sockets.
  • Frontend con Next.js + React para renderizado rápido y SEO.
  • Panel CMS para gestión de productos, clientes y pedidos.
  • Chat IA integrado para atención y soporte al cliente.

3. Funcionamiento General

Flujo del cliente y administrador:

  • El cliente entra al sitio → se muestran los productos destacados y promociones.
  • Agrega productos al carrito → persistente en sesión o localStorage.
  • Finaliza compra en checkout → se crea el pedido en la base de datos.
  • El administrador recibe notificación en tiempo real vía Socket.io.
  • Stock y pedidos se actualizan instantáneamente en frontend y CMS.
  • Chat IA disponible para consultas, sugerencias y soporte al cliente.
  • CMS permite gestionar clientes, productos, stock y generar reportes.

4. Ventajas de esta Arquitectura

  • Escalable: se puede replicar la tienda para otras marcas sin cambiar la arquitectura.
  • SEO friendly: Next.js permite renderizado en servidor para mejor indexación.
  • Tiempo real: stock y pedidos se actualizan instantáneamente.
  • Modular: frontend, backend y chat IA separados para fácil mantenimiento.
  • Interactividad: Chat IA y notificaciones en tiempo real aumentan la experiencia de usuario.
