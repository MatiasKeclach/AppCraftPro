Guía de despliegue de AppCraftPro

1. Preparar el repositorio en GitHub

Ya subiste tu proyecto a GitHub. Verifica que tengas:

  • Todos los archivos del proyecto.
  • Archivo .gitignore para excluir node_modules y archivos temporales.
  • Token personal para autenticar los pushes desde consola.
git init
git add .
git commit -m "Versión inicial"
git branch -M master
git remote add origin <URL_DEL_REPO>
git push -u origin master
Tip: Si tienes problemas de autenticación, usa el token de GitHub en lugar de la contraseña.

2. Crear la app en Render

  • Entra a Render y crea una cuenta si no la tienes.
  • Selecciona Web Service y conecta tu cuenta de GitHub.
  • Elige tu repositorio AppCraftPro y la rama master.
  • En Build Command coloca npm install.
  • En Start Command coloca node app.js o tu script principal.
  • Selecciona un entorno de Node.js compatible (ej: 22.x).

3. Configurar la base de datos

  • Si usas SQLite3, sube el archivo appcraftpro.db al repositorio o usa Persistent Disk en Render.
  • Asegúrate de que la ruta en db.js coincida con la ubicación real en Render.
  • Si quieres iniciar limpia, crea un script como initDB.js para generar todas las tablas al iniciar.
Tip: Render permite variables de entorno, úsalas para la configuración de paths y contraseñas si luego agregas servicios externos.

4. Despliegue y pruebas

  • Render construirá la app automáticamente al detectar package.json.
  • Revisa los logs de construcción y ejecución para asegurarte que no hay errores.
  • Prueba el login con tu superadmin y verifica que la base de datos esté accesible.
  • Si agregas nuevas dependencias, haz un commit y push a GitHub; Render actualizará automáticamente.

5. Tips finales

  • Mantén tu node_modules en .gitignore, Render lo instala automáticamente.
  • Sube los archivos de configuración esenciales (DB, scripts de inicialización, .env si lo usas).
  • Para actualizaciones, solo hace falta push al repo y Render redeploy.
  • Si tu app crece, considera usar Postgres o MySQL en Render en lugar de SQLite para persistencia segura.
