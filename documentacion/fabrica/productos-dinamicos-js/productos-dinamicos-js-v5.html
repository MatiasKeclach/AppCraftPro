Fábrica de Tiendas Virtuales

🤖Inteligencia Artificial

La IA automatiza la creación de tiendas virtuales, generando productos y menús dinámicos.

Cómo funciona: un JSON de productos y categorías se procesa para renderizar automáticamente la tienda.

🔑API

Permite crear tiendas automáticamente y gestionar productos, categorías y usuarios.

fetch('https://midominio.com/api/tienda', {
  method: 'POST',
  headers: {'Authorization': 'Bearer API_KEY'},
  body: JSON.stringify({nombre: 'Tienda Demo'})
});

🛡️Roles y Permisos

  • Owner: Acceso total, genera API Key, crea tiendas y administra roles.
  • Superadmin: Gestiona tiendas, usuarios y contenidos, sin API Key del owner.
  • Admin: Gestión de contenido y operaciones dentro de tiendas asignadas.
  • Cliente: Usuario consumidor de la tienda.

Ejemplo de verificación de rol:

if(user.role !== 'owner') {
  console.log('No tienes permisos para ejecutar esta acción');
  return;
}
